# Demoscene Interactive Book
An interactive tool built on HARFANG 3D to show what demoscene is about

## How to run it?

- Windows only, sorry folks :/

### By _cloning_ the repository

- `git clone https://github.com/astrofra/demoscene-interactive-book.git`
- run `2-built.bat`
- run `3-start.bat`

### By downloading the _release_ version

- Download the release from here : https://github.com/astrofra/demoscene-interactive-book/releases
- Unzip it
- run `_start.bat`

## How to use it?

- check here : https://github.com/astrofra/demoscene-interactive-book/blob/main/app/_Readme.md
