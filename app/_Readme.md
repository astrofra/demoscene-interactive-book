# Présentation interactive. Requiert un PC sous Windows 10/11 avec GPU.

## Démarrage et Fermeture

- Pour lancer la présentation, double-cliquez sur `_start.bat`.
- Pour quitter, appuyez sur `ESC`.

## Navigation

### Naviguer au clavier (par défaut)

- Pour passer d'une slide à l'autre : :arrow_left: & :arrow_right:
- À l'intérieur d'une slide, pour passer d'un sujet à l'autre : :arrow_up_small: & :arrow_down_small:
- Slide 02 : pour "piloter" dans la Carverne, utilisez `Z`, `Q`, `S`, `D`

### Naviguer avec un contrôleur (PlayStation Dual Shock 4)

- Pour passer d'une slide à l'autre : :black_square_button: & :red_circle:
- À l'intérieur d'une slide, pour passer d'un sujet à l'autre : :small_red_triangle: & :x:
- Slide 05 : pour faire tourner les packshots, utilisez le stick analogique droit et les gâchettes.
- Slide 06 : pour piloter la voiture, utilisez le stick analogique droit et les gâchettes.
- Slide 10 : pour piloter la caméra sur la simulation lumineuse, utilisez le stick analogique droit et les gâchettes.
- Slide 12 :
  - Pour la galerie photo : utilisez les gâchettes.
  - Pour le graphique représentant le processus : utilisez le stick analogique droit et les gâchettes.
